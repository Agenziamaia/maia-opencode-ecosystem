3.  **Visual Blindness**: Vibe Kanban shows *Tasks*, but we have no dashboard for *Neural Activity* (Which agent is thinking? Who is overloaded?).
    - *Improvement*: A "Pulse" Dashboard showing real-time token flow and agent states.

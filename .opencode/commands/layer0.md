This command triggers the Layer 0 Generation Script.

## What it does
1.  Analyzes the current ecosystem.
2.  Extracts Universal Intelligence (Agents, Giuzu, Skills).
3.  Generates a new folder on Desktop named `MAIA_Layer0_vYYYY-MM-DD.N`.
4.  Sanitizes the new folder (removes logs/sessions).

## When to use
- When you want to save your current "Brain State" as a template.
- When you want to start a completely fresh project but keep your agents smart.
- When you want to move to a new machine.

## Execution
Run this bash command:
```bash
./.opencode/scripts/generate_fresh_layer0.sh
```
